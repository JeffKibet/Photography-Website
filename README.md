Lens and Light Photography

Description

Lens and Light Photography is a simple photography portfolio website designed to showcase beautiful images and highlight the art of capturing moments through light and perspective. The website provides a clean and modern layout where visitors can explore different photographs, learn about the photographer, and easily navigate through the site.

This project was created as part of learning web development fundamentals, focusing on building a structured and visually appealing website using HTML and CSS.
